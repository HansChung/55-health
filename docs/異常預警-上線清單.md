## 🩺 常見問題

| 症狀 | 原因 | 修法 |
|------|------|------|
| curl 回 `Unauthorized` | CRON_SECRET 沒設或打錯 | 檢查 Vercel 環境變數 + Redeploy |
| `alerts_fired: 0` | 沒有符合條件的長輩/家人 | 確認家人連結 status=accepted、permissions.alerts=true |
| 沒收到信 | RESEND_API_KEY 沒設 / nuan55.com 沒 verified | 檢查 Resend Domains 綠勾 |
| 信進垃圾信夾 | 第一次寄信正常現象 | 按「非垃圾信」 |
| 重複觸發沒再寄信 | 24h 防重複機制（正常） | 等 24h 或刪 alerts 表那筆記錄再測 |
